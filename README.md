#  TP noté 

## Installation et lancer le projet
Installation
```
npm install
```


Lancer le serveur "back"
```
npm run server:back
```

lancer le serveur "front"
```
npm run server:front
```


## Ennoncé 

C'est un mini projet regroupant le Polling, Long-Polling et SSE. 
Ce projet à pour but de récupérer des 3 manières différentes les messages qu'on envoie depuis le client
